# 分页功能说明 — 每一页能看什么、决定什么

云端面板 `miru-ops-dashboard.streamlit.app` 顶部有 3 个大类切换：「🅰️ 财务结果」「🅱️ 会员价值」「🅲 代理 / 渠道」。每个大类底下有几个分页。

---

## 🅰️ 财务结果

### 1. 经营总览

**资料来源**：`raw_platform_report` + `raw_finance_report`

**能看什么**：
- 公司输赢、有效投注额、实际总存款、存提差、注册数、首存转化率
- 每日 / 每月趋势线
- 真实平台净盈利（系统输入 vs 手动输入对照）

**适合什么时候打开**：每周看一次大盘走势、汇报老板月度业绩。

---

### 2. 红利分析

**资料来源**：`raw_bonus_report`

**能看什么**：
- 红利总金额、笔数、领取会员数、人均
- 代理相关红利 vs 真实会员红利分流（你自己挑哪些活动名称归代理）
- 按活动名称 / 红利类型分布

**适合什么时候打开**：审核月度红利账单、找出哪些活动占成本最高。

---

### 3. 红利 ROI & 代理质量 ⭐ (6/4 新)

**资料来源**：`raw_bonus_report` + `raw_agent_report`

**能看什么**：

**Tab A — 红利 ROI**：
- 红利总成本 vs 同期公司收入（看红利成本占比）
- 每日红利成本 vs 公司收入 双轴趋势
- 红利标题 Top 30（哪些活动最贵 / 候选砍）
- 🚨 红利党会员清单（单日领 ≥3 笔的会员 + 累积红利）
- 流水门槛分析（看哪些红利免流水容易被套）
- VIP 等级 × 红利成本

**Tab B — 代理质量**：
- 代理质量散点图（拉新人数 vs 公司收入，颜色 = 红利依赖度）
- 优质代理 Top 20
- 🚨 红利党代理 Flag（红利依赖度 > 50% + 公司收入 ≤ 0）
- 亏损代理 Bottom 20

**适合什么时候打开**：每月 1 次砍/升活动的决策、谈代理续约前。

---

### 4. 代理佣金 & 退成

**资料来源**：`raw_agent_commission_*` + `raw_agent_settlement_*`

**能看什么**：
- 代理团队佣金月度统计
- 单笔佣金明细
- 介绍人退成（介绍人佣金分成）

**适合什么时候打开**：月度佣金核算 / 处理代理佣金争议。

---

## 🅱️ 会员价值

### 5. 会员结构 & ARPU

**资料来源**：`raw_member_report`

**能看什么**：
- 总会员 / 活跃会员 / 新会员
- ARPU（每用户均收入）/ 公司收入
- VIP 等级分布
- TOP 20 投注占比（集中度）
- 次月活跃留存率 / 次月存款留存率

**适合什么时候打开**：评估 VIP 政策效果、看会员集中度风险。

---

### 6. 投注分析

**资料来源**：`raw_bet_detail_2026_04` 等月度注单明细

**能看什么**：
- VIP × 类型 交叉表（对照「真人 1.15% / 体育 1.5% / 电子 1.5%」holding ceiling）
- TOP 100 大户流水集中度
- 走盘 / 和局退本金 / 不计入活动玩法的扣减

**适合什么时候打开**：审让利 cap / 大户监控。

---

### 7. 客服分析 ⭐ (6/5 新)

**资料来源**：`raw_cs_conversations`（包网商每月 xlsx）

**能看什么**：

- 对话总数 / 客服数 / 涉及会员 / 平均响应 / 平均总时长
- 满意度分布（非常满意 / 满意 / 不满意 / 非常不满意 / 未评价）
- **客服效率排行**：每个客服 接待量 / 响应秒数 / 总时长 / 对话回合 / 好评率
- **小时分布 + 星期分布**（看排班是否合理）
- 服务主题 Top 20
- 🚨 **非常不满意 / 不满意 案件 deep dive**：
  - **主因自动抽取**（服务主题 > 评价内容关键词 > 对话内容关键词）
  - 主因分布 + 水平 bar chart
  - 案件表格 + 「贴对话 ID 直接定位」+ 下拉选案件
  - 点开看完整对话全文 + KPI + 客户评价
- 会员侧 VIP 等级 × 问题量
- 对话关键词热点 Top 25

**适合什么时候打开**：每月看一次客服质量 + 找出包网商客服服务漏洞。

---

### 8. 实时波动 & DAU

**资料来源**：`raw_realtime_bet`（即时注单）

**能看什么**：
- DAU 走势
- 时段投注热度
- 异常波动监控

**适合什么时候打开**：日常监控大单 / 异常下注。

---

## 🅲 代理 / 渠道

### 9. 代理团队 & 渠道

**资料来源**：`raw_agent_report` + `raw_promotion_report`

**能看什么**：
- 代理团队总数、首存数、累计投注
- 渠道维度的拉新、首存、留存
- 一级 / 二级 / 三级 / 四级层级分析

**适合什么时候打开**：月度代理体系健康检查。

---

### 10. 代理 × 会员 明细 ⭐ (6/4 新)

**资料来源**：`raw_member_report`

**能看什么**：

- **代理总览**：每个代理 会员数 / 累积投注 / 公司净 / 红+返/投注% / holding% / 套利户数
- 顶部 KPI（代理总数 / 会员总数 / 累积投注合计 / 套利特征户数）
- 排序：可按累积投注 / 公司净 / 会员数 / 红返% / 套利户数
- 搜索：输代理名快速过滤
- **单查代理 / 会员**：
  - 输代理名 → 看该代理底下全部会员的 KPI
  - 输会员账号 → 直接定位单一会员 + 显示月度趋势
  - 备用下拉框选 top 200 大代理
- 🚨 **系统已 tag 但状态启用的会员清单**（套利 / 高风险 / 多平台 / 跨平台对打 / 骗分 / 专业玩家 / 软件投注）
- 🔴 套利特征 + 启用 + 公司净亏损 = 强烈建议处置清单

**适合什么时候打开**：风控案件复查、代理质量谈判、决策处置套利户。

---

### 11. 游戏 & 场馆

**资料来源**：`raw_game_report` + `raw_game_report_venue` + `raw_game_analysis`

**能看什么**：
- 每个游戏当月投注 / 输赢 / 盈余率
- 每个场馆当月统计
- 场馆 mix 分布（找最赚 / 最赔 的场馆）

**适合什么时候打开**：场馆配置调整 / 跟包网商谈场馆维护。

---

## 通用使用提示

1. **每个分页顶部都有「开始日期 / 结束日期 / 快速切换月份」筛选器**，可以指定时间范围。
2. **大部分图表 hover 可以看详细数字**。
3. **表格可以点栏位排序、可以下载 CSV**（右上角小按钮）。
4. **5 月数据 cache 5 分钟**，刚 import 完要等 5 分钟才能在面板看到最新。
5. **看不到新月份数据** → 看 `2_操作手册_每月上报.md` 的故障排除。
