# 资料夹结构说明

整套面板的本机资料夹结构：

```
dashboard/                              本资料夹（运营数据面板完整安装包）
│
├── 1_首读_README.md                  ← 总览 + 快速上手
├── 2_操作手册_每月上报.md             ← 每月报表来了怎么传
├── 3_分页功能说明.md                 ← 面板每个分页能看什么
├── 4_资料夹结构说明.md               ← 本文档（你正在看的）
│
├── dashboard.py                       面板 Streamlit 源码（云端 + 本机共用）
├── import_tool.py                     报表导入工具源码
├── requirements.txt                   Python 套件清单
├── .gitignore                         Git 忽略清单（含 secrets.toml）
├── .streamlit/                        Streamlit 设定（本机敏感档不入 git）
│
├── 启动运营面板.bat                   双击 → 开启本机版面板（localhost）
├── 启动运营面板_背景.vbs              双击 → 后台启动（不弹 cmd 视窗）
├── 报表导入工具.bat                   双击 → 进入导入菜单
│
├── 月度报表/                          ⬇ 每月报表放这里
│   └── (8 张 xlsx / csv / zip)
│
├── 红利记录/                          ⬇ 红利 CSV 按月份分子资料夹
│   ├── 202601/
│   ├── 202602/
│   ├── 202603/
│   ├── 202604/
│   └── 202605/   ← 5 月示范（已就位）
│
├── 客服对话/                          ⬇ 客服对话 xlsx 按月份分子资料夹
│   └── 202605/   ← 5 月示范（已就位）
│       └── 5月客服对话.xlsx
│
└── 已清洗/                            import 工具自动产出的清洗后档案
                                      （可定期清理 / 不影响 BQ 资料）
```

## 每个东西做什么的

### 你会用到的（双击 / 编辑）

| 名称 | 类型 | 用途 |
|---|---|---|
| `1-4_*.md` 四份 | 说明文档 | 阅读用，看完你就知道整套怎么用 |
| `启动运营面板.bat` | 双击 | 在本机跑 Streamlit 面板（不连云端） |
| `启动运营面板_背景.vbs` | 双击 | 静默启动面板 |
| `报表导入工具.bat` | 双击 | 跑 import_tool 上传报表 |
| `月度报表/` | 资料夹 | 月度 8 张报表放这里 |
| `红利记录/YYYYMM/` | 资料夹 | 该月红利 CSV 放这里（要按月分） |
| `客服对话/YYYYMM/` | 资料夹 | 该月客服 xlsx 放这里（要按月分） |

### 工程师才动的（请勿手动改）

| 名称 | 类型 | 用途 |
|---|---|---|
| `dashboard.py` | 源码 | 面板逻辑 |
| `import_tool.py` | 源码 | 导入工具逻辑 |
| `requirements.txt` | 文字 | Python 套件清单 |
| `.gitignore` | 文字 | Git 忽略清单（防 secrets 入 repo） |
| `.streamlit/` | 资料夹 | Streamlit 设定（含 secrets.toml） |
| `已清洗/` | 资料夹 | import 副产物 |
| `__pycache__/` | 资料夹 | Python 自动产出 |

## 资料夹命名规则

### 月份资料夹一律 6 位数字 `YYYYMM`

| 月份 | 资料夹名 |
|---|---|
| 2026 年 1 月 | `202601` |
| 2026 年 5 月 | `202605` |
| 2026 年 12 月 | `202612` |
| 2027 年 1 月 | `202701` |

不要写 `2026-05` 或 `2026年5月` 或 `2026.05`——import 工具的正则只认 `YYYYMM`。

### 文件命名（亚博包网商提供的）通常已经对，不要动

例外是 **TOP 报表**——它可能叫 `top.xlsx`，要改成 `top-report 202506.xlsx` 才会被识别成 5 月 TOP 报表（详见 `2_操作手册_每月上报.md` 坑 1）。
